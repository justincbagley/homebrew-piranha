# homebrew-piranha

A Homebrew tap with (Ruby) formula for installing the following:

- [PIrANHA](https://github.com/justincbagley/piranha)


#### Add this tap
`brew tap justincbagley/piranha`


#### Install
- PIrANHA: `brew install piranha`
